# Practica04-Mi-Correo-Electr-nico
La pagina HTML se maneja en 3 partes.

LOGIN:
Contiene la ventana de login y creaciond de nuevos usuarios, donde los archivos son publicos.

PUBLIC:

Contiene la parte de los usuarios en general
*Enviar Mensajes
*Editar Perfil
*Cambiar Contraseña
*Bandeja de Mensajes
*Mensajes Enviados.
*Abrir Mensajes

ADMIN:

Contiene la parte de los usuarios en general
*Listar Mensajes
*Editar Usuarios
*Cambiar Contraseñas
*Bandeja de Mensajes.
*Abrir Mensajes.
